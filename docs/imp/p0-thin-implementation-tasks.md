## 初期実装方針

- 初期は `apps/web` に閉じる。
- DB、API、認証、外部連携は作らない。
- 状態はブラウザ内の mock data として扱う。
- P0 薄く実装 1 版では、AI 整理結果は全件確認待ちにする。
- UI は `shadcn/ui`、Plane、OpenProject、Leantime の情報設計を参考にするが、依存導入で足止めしない。
