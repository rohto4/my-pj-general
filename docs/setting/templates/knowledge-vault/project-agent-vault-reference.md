# Project AGENTS.md Vault Reference

各開発PJの `AGENTS.md` に、必要に応じて次を追加する。

```markdown
## 横断ナレッジ vault

- 横断ナレッジ vault: `G:\knowledge-vault`
- PJ固有の実装状態、Issue、PR、作業ログはこのPJ内を正本にする。
- 複数PJで再利用できる知識、判断、根拠資料、AIセッション要約だけ `G:\knowledge-vault` に保存する。
- `G:\knowledge-vault\memory\l1-triggers.md` は想起入口として扱う。
```

## 使い分け

- PJ内: 実装、タスク、PR、Issue、PJ固有ルール。
- `G:\knowledge-vault`: 横断知識、再利用知識、根拠、LLM記憶。
