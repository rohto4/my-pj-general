# my-LLMwiki Project Context

## 目的

この vault は、`G:\knowledge-vault` を材料にした製本済み知見を読むための場所。

## 前提

- 正本は `G:\knowledge-vault`
- compile は `G:\knowledge-vault\scripts\compile-llmwiki\weekly-compile-to-llmwiki.ps1`
- この vault では topic, case, entity, question を中心に読む

## GitHub 運用

- ある程度進んだら `rohto4` アカウントで GitHub に commit / push しながら進める
- 想定 remote: `https://github.com/rohto4/my-LLMwiki.git`
- 週次 compile と並行して、意味のある単位で定期的に commit / push する
