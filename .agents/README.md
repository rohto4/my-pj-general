# .agents

このディレクトリは、PJ で使う agent skill や補助設定を置く。

docs 更新ルールや knowledge-vault 更新判断の正本はここには置かない。

- PJ docs 更新判断: `docs/guide/docs-management-rules.md`
- 横断ナレッジ更新判断: `G:\knowledge-vault\knowledge-vault-write-policy.md`

新しい skill を追加する場合も、docs 更新の判断表を skill 側へ重複コピーしない。
必要な場合は `docs/guide/docs-management-rules.md` を参照させる。
